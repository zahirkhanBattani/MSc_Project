# AI Risk Assessor

## Overview
This project implements an AI-based tool to assess project risks in generic projects.

## Environment
- Python 3.11 (ai_risk conda environment)
- Libraries: numpy, pandas, matplotlib, scikit-learn, jupyter, spacy, nltk

## Structure
- notebooks/: Jupyter notebooks for experiments
- scripts/: Clean Python scripts
- data/: Project datasets
- figures/: Generated plots
- reports/: Written reports
